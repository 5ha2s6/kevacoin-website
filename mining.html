Mining

Kevacoin uses Proof-of-Work as its consensus method. The hashing algorithmn is Cryptonight R (variant 4). Cryptonight based cryptocurrencies (e.g. monero, bytecoin) have different header structure than Bitcoin/Litecoin. Therefore, the existing Cryptonight miners (e.g. XMRig) are incompatible with Kevacoin. We provide reference implementations for CPU, AMD and Nvidia GPU based on ccminer.

Mining Software

Even though Kevacoin is based on Bitcoin/Litecoin, we slightly modified the header structure to make it compatible with Monero mining software. As a result, one can use any Monero mining software to mine Kevacoin (e.g. XMRig or XMR-Stak).

Solo Mining

We provide a reference implementation of Kevacoin mining stratum keva-stratum based on monero-stratum. It is easy to build from source code on Linux and macOS. For Windows, we provide a pre-built binary.

Mining Pools

We provide a reference implementation of Kevacoin mining pool based on node-cryptonote-pool.
