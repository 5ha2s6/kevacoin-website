The Blockchain with Key-Value Data Store

Kevacoin is a cryptocurrency based on blockchain technology, featuring a decentralized Key-Value Store. Use it for payments, as well as secure data storage and distribution, without a central authority or database.

What is Kevacoin?

Kevacoin is a decentralized open source key-value data store based on the Bitcoin/Litecoin, and influenced by Namecoin.

What does it do?

  • Securely store keys and their values. No hard limits on the number of keys.
  • Maintain network-unqiue namespaces. Keys are grouped under namespaces to avoid name conflicts.
  • Transact the digital currency kevacoins (KVA).

What can it be used for?

As a decentralized key-value database, it can be used to store data for all kinds of applications, such as social media, microblogging, public identity information, notary services. Acccess the data store is secure and easy, without the need of writing and deploying smart contracts. The data is decentralized while the application logic can be developed off the blockchain using any programming languages.

Contribute

Kevacoin is an open source project and it is not funded by any companies or foundations. you are invited to make Kevacoin better!
